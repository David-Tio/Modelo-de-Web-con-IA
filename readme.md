# Modelo de Web para Generación de Presupuestos por IA e Imagen (Modelo Básico)

Este proyecto consiste en el desarrollo de una aplicación web capaz de generar presupuestos automáticamente utilizando inteligencia artificial, basándose tanto en texto como en imágenes proporcionadas por el usuario.

El objetivo es simplificar y automatizar el proceso de creación de presupuestos para diferentes sectores (construcción, mantenimiento, diseño y otros), analizando la información suministrada y produciendo un resultado claro, rápido y preciso.

---

## Características principales

- IA integrada para interpretar texto, descripciones e instrucciones del cliente.
- Procesamiento de imágenes, permitiendo subir fotos del producto, área o elemento a presupuestar.
- Generación automática de presupuesto detallado.
- Descarga del presupuesto en PDF o visualización en pantalla.
- Interfaz web intuitiva y fácil de usar.
- Sistema modular para permitir mejoras y nuevas funciones.
